SPAM Optimisation

At Telstra, I was tasked with creating a system that identified re-occuring SPAM being sent over our network - and informing the provider it is coming from. I.e. Optus

I was provided the front-facing website of the current system that picked up SPAM messages and told to figure it out! Firstly, I utilised scraping software and built third-party keys with get requests to pull the recorded SPAM for a given hour. I recorded this on a local linux server, i then ran some simple Python to check if the SPAM was being sent from a number for more than 8 hours -> If it was I compiled it into a list -> Then every morning and afternoon I sent out automatic Emails to the relevent providers for the given time period to inform them of the Spamming numbers. This is still operational at Telstra, as I cc'd my personal Email! But I can't provide the code as per Telstra's employee requirements.

Swyftx: Boosting team efficiency

At Swyftx, I worked as an online customer service rep, the role was quite competitive. To increase the number of people I could assist, while maintaining my service quality I implemented a number of macros.

I initially used AutoHotKey, but found it a little inconsistent as it's colour detection wasn't ideal and would click the wrong parts of the screen.

So I then used Python, as it's inbuilt image detection meant it would always click the right part of the screen regardless of how big it was/where it was. This worked brilliantly and improved my efficiency by 15% and teammates efficiency by 10-15% depending on the person.
